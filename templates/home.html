Welcome to the Blockchain Transaction System

Commands

Our blockchain is built as an API so each of the below commands is just an API endpoint with an associated python function. To use the commands simply append the one you'd like to use to the end of the address. (i.e. http://127.0.0.1:5000/mine)

GET Commands

The below commands are get commands and can be appended directly upon the current address in the address bar and the output will bbe displayed in the browser.

/mine This command mines a new block and adds it to the chain.

/chain This command prints out the entire chain

/nodes/resolve This command runs the concensus algorith =m and updates your chain to be the most up to date.

POST Commands

The below commands are require a HTTP requester of some sort. Options include cURL from within the terminal or Postman.

/transactions/new This command mines a new block and adds it to the chain.

Example Input:

{
"sender":"8594924d0d2f4f6381a4656c0998b7ac",
"receiver":"someone_else",
"value":1,
"note": "first transaction"
}

/nodes/register This command reigisters new nodes to commincate with the blockchain.

Example Input:

{
"nodes": ["http://10.100.157.70:5000"]
}
